教你通过 Shortcut 捷径实现控制电脑开关机

发表于 2018-09-18 | 主题为 写点教程

1、依次进入 Mac 电脑的「偏好设置 - 共享」，勾选「远程登录」的开关。

2、按住键盘上的 Option 并点击右上角的 Wi-Fi 图标，获取你的 IP 地址。

3、获取控制电脑开关机的 Shortcut 链接，在自带浏览器中 戳这里添加。

4、点击「…」，编辑 ShortCut，并修改为刚才你获取的 IP 地址、填入你电脑的账户名、和密码。

5、之后点击 ShortCut 即可运行了。
